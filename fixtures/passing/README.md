# Passing CLI

A sample npm CLI that passes all release evidence checks.

## Target User

CLI developers who need a working example of a npm package with all required release files.

## Quickstart

Install:
```bash
npm install passing-cli
```

Run the CLI:
```bash
npx passing-cli
```

## Expected Output

When you run `npx passing-cli`, you should see:
```
Passing CLI is working!
Usage: passing-cli --help
```
