# Contributing to Passing CLI

We welcome contributions! Here is how you can help:

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/passing-cli.git`
3. Install dependencies: `npm install`
4. Create a feature branch: `git checkout -b feature/my-new-feature`

## Code Style

This project uses ESLint and Prettier. Run `npm run lint` and `npm run format` before submitting a PR.

## Submitting Changes

1. Ensure all tests pass: `npm test`
2. Push your branch: `git push origin feature/my-new-feature`
3. Open a Pull Request against the `main` branch.

## Code of Conduct

This project adheres to the Contributor Covenant Code of Conduct. By participating, you are expected to uphold this code.
